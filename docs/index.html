About Sapresis

Sapresis (Comprehensive System for Health Service Provision) is designed for use by health service providers. It enables the management of various entities related to institutional employees, consultations, clinics, patients, prescriptions, billing, and medical professionals.


Project Description

Sapresis is a system developed to manage the information of staff and patients within a healthcare institution. It allows users to:

  • Consult and manage institutional staff information (ID, name, and contact details).
  • Assign consultations, clinics, and patients to the respective doctors.
  • Retrieve dynamic information such as medical prescriptions, billing details, and patient data.

Technologies Used

  • Spring Boot (Backend)
  • MySQL (Database)
  • React (Frontend)
  • HTML, JavaScript, Bootstrap 5


Installation Guide

Clone the Repository

To install and run Sapresis locally, follow these steps:

  1. Clone the repository:
    2. git clone https://github.com/MauricioMonroy/sapresis.git
  2. Navigate to the project folder:
    3. cd sapresis

Backend Setup

  1. Ensure a MySQL database is configured.
  2. Update the application.properties file with your database credentials.
  3. Run the Spring Boot application using Maven or your favorite IDE.

For details on the project structure, consult the ARCHITECTURE document.

For information about the API and how to interact with it, check the API DOCUMENTATION guide.

Frontend Setup

  1. Navigate to the frontend directory.
  2. Run npm install to install dependencies.
  3. Start the application with npm start for development mode or npm run build for production mode.

For detailed installation instructions, refer to the INSTALLATION guide.

For instructions on how to use and operate the application, check the USAGE guide.
