El Equipo

Un proyecto con éxito necesita a mucha gente en varios roles. Algunos miembros escriben el código o la documentación, mientras que otros son importantes como testers o enviando parches y sugerencias.

El equipo está compuesto por Miembros y Colaboradores. Los Miembros tienen acceso directo al código fuente del proyecto y hacen evolucinar el código de forma activa. Los Colaborares mejoran el proyecto mediante el envío de parches y sugerencias a los Miembros. El número de Colaboradores del proyecto no está acotado. Implícate hoy. Todos los colaboradores del proyecto son muy valorados.
